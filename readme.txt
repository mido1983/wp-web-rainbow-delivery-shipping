Объединённый плагин: Custom Delivery Date + Custom City Shipping

== Description ==
Этот плагин служит примером объединения двух плагинов в один.
Поддерживает настройку даты доставки и логики доставки по городам.


== Installation ==
1. Загрузите папку плагина `web-rainbow-delivery-shipping` в `/wp-content/plugins/`.
2. Активируйте плагин через меню «Plugins» в WordPress.
3. Настройте плагин на странице «Delivery & Shipping».
